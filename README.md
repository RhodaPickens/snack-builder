# Snack Builder

A simple React app that suggests snack options based on calorie budget and hunger level.

![Snack Builder Screenshot](./public/screenshot.png)

## Features

- Calorie input
- Hunger level selector
- Snack recommendations based on how filling they are
- Simple UI for quick decisions

## Tech Stack

- React
- JavaScript
- CSS

## Running the Project Locally

npm install  
npm run dev
